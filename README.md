# Deepak Vocablaury 📚

**A Level-based English to Hindi Vocabulary Learning Web App**

### About the Project
Deepak Vocablaury is a simple and clean web app to learn English words with Hindi meaning and example sentences. 
Made for students, beginners, and job seekers to improve their English vocabulary daily.

### Features ✨
- **Level 1 - Easy 🟢** : 10+ Basic words like Apple, Book, Cat
- **Mobile Responsive** : Works perfectly on phone and PC
- **Hindi Meaning + Example** : Easy to understand and remember
- **Clean UI** : Beautiful gradient design

### Tech Stack 💻
- HTML5
- CSS3
- JavaScript
